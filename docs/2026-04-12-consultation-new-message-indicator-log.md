# 2026-04-12 问诊新消息提醒条开发记录

## 本次完成

- 患者端问诊详情页增加“收到新消息，跳到最新”提醒条。
- 医生端问诊详情页增加“收到新消息，跳到最新”提醒条。
- 提醒条仅在以下条件下出现：
  - 页面自动刷新拉到了新消息。
  - 当前用户没有停留在消息列表底部。
- 用户点击提醒条后会立即跳转到最新消息，并清除提醒状态。
- 当用户手动滚动回到底部时，提醒条会自动消失。

## 交互细节

- 初次打开问诊详情仍会自动定位到最新消息。
- 若用户正在查看旧消息，自动轮询只同步数据，不打断阅读位置。
- 有新消息到来时，通过轻提醒条提示，而不是强行滚动到底部。
- 患者自己发送消息、提交恢复更新、提交检查结果后，会主动回到最新消息，避免把自己刚发出的消息也当作“新提醒”。
